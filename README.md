# IIOT

A basic project for mimicking an IIOT environment. 

## Description

A basic project for mimicking an IIOT environment. 
   

## Getting Started

### Dependencies

* MQTTnet library
* SignalR
* EntityFrameworkCore
* SqLite


### Executing program

* The following command can be executed at the solution level, to spin up the Broker, Simulator, API, Consumer Services and the UI.
```
docker-compose up --build
```
